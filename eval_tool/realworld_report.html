Real-world validation — structural measures

The benchmark's second source. The structural measures run on real, naturally-occurring code: methods Pylint flagged with each smell (CodeSmellData 1.0/2.0, PySmell) versus a pool of 1000 real clean methods from the same GitHub corpus. Detection strength = unpaired Cohen’s d (smelly vs clean), shown against the injected (paired) value. Similarity measures need a clean twin, so they do not apply to real code.
